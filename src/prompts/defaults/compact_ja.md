# 制約
- 登場人物・関係・状態・直前の出来事を漏らさず残す
- 設定（場所・時間・所持品・体勢・衣服の状態など）は省略しない
- 文体は本文ではなく「あらすじ」として書く（地の文の三人称・常体）
- 800〜1500 文字程度
- 前置き・後書き・メタ発言なし。あらすじ本文のみ出力（出力全体が新しい `<!-- DIGEST: ... -->` ブロックの中身として使われるため、`<!-- DIGEST: -->` ラッパー自体は書かないこと）

# コメント記法について
本文中の `<!-- LABEL: 中身 -->` 形式の HTML コメントは作者向けメタ情報。`<!--` 直後の大文字ラベルで種別を判別する。中身はコロン直後の空白から続く（短ければ1行、長ければ改行を挟んで複数行）。

- `NOTE` … **著者注**。作者からの指示。
  - 要約に取り込まず、あらすじ本文の対応する位置にそのまま残すこと（後続の生成にも作者の指示を引き継ぐため）。
- `DIGEST` … **既出のあらすじ要約**（前回の圧縮結果）。
  - 中身を既に起こった出来事として読み、新しい要約に統合すること（コメントブロック自体は剥がし、地の文として吸収してよい）。
